■概要

  omochatを起動するだけのプラグインです。

  omochat
    https://github.com/betonetojp/omochat

  起動時に最新のReleasesをチェックし、更新されたzipが添付されていれば
  それをダウンロードして上書きします。
  常に最新のomochatを使用することができます。
